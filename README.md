## 🎯 About

One-UI is an advanced VPN management panel for Outline VPN servers with support for dynamic keys and multi-user authentication.

## ✨ Features

- 🎨 Modern Dashboard
- 🖥️ Server Management
- 🔑 Access Keys
- ⚡ Dynamic Keys
- 📊 Analytics
- 🌐 Bilingual (EN/MY)
- 📱 Responsive Design

## 🚀 Quick Start

\`\`\`bash
git clone https://github.com/sankahchan/One-UI.git
cd One-UI
npm install
cp .env.example .env
nano .env
npx prisma generate
npx prisma migrate deploy
npm run setup
npm run build
npm start
\`\`\`
